## ✨ Features
- **Live market overview** – top cryptocurrencies by market cap with price, 24h change, market cap, volume and supply.
- **Exchange directory** – searchable, filterable list of major exchanges (CEX & DEX) with key stats and detailed trading‑pair view.
- **Detailed exchange pages** – description, year established, 24h volume, type, and a searchable table of trading pairs.
- **Automatic data refresh** – timestamps update every minute; manual refresh button on the dashboard.
- **Responsive UI** – sidebar navigation, dark mode support, and a clean layout inspired by the official Nuxt dashboard template.
- **Reusable components** – `StatCard`, `NavLink`, `Icon` for rapid UI development.
- **Type‑safe data handling** – composable `useCryptoData` with TypeScript interfaces and formatting helpers.

---

## 🛠️ Tech Stack
- **Framework:** Nuxt 4 (SSR/SSG) with TypeScript
- **UI Library:** `@nuxt/ui`
- **Styling:** Tailwind CSS (custom utility classes & dark mode)
- **Icons:** `lucide-vue-next`
- **Data source:** [CoinGecko API](https://www.coingecko.com/en/api) (public, no API key required)

## 🚀 Getting Started
### Prerequisites
- Node >= 20
- `pnpm` (recommended) – install with `npm i -g pnpm`

### Installation
```bash
# Clone the repository (or copy the project folder)
git clone <repo‑url>
cd <project‑folder>

# Install dependencies
pnpm install
```

### Development
```bash
pnpm dev   # Starts the Nuxt dev server at http://localhost:3000
```
The dashboard automatically adapts to light/dark mode based on your system settings.

### Production Build
```bash
pnpm build   # Generates a production‑ready build
pnpm preview # Preview the built app locally
```

## 📡 API Usage
All data is fetched client‑side from CoinGecko’s public endpoints – no authentication required. The composable handles:
- `/coins/markets` – top‑ranking crypto market data
- `/exchanges` – list of exchanges with volume & type
- `/exchanges/{id}` – detailed exchange info
- `/exchanges/{id}/tickers` – trading pairs for a specific exchange
- `/search` – cryptocurrency search
- `/coins/{id}` – full crypto details (available for future extensions)

Rate limits are generous for development; for production consider a paid plan or implement caching.

## 🎨 Customisation
- **Theme colors** – edit `nuxt.config.ts` under `ui.colors`.
- **Tailwind extensions** – modify `tailwind.config.ts` to add fonts, spacing, or plugins.
- **Components** – add more Nuxt UI components (charts, tables) to enrich the dashboard.

## 🐞 Troubleshooting
- **Tailwind errors** – ensure `tailwindcss` and `autoprefixer` are installed (they are listed in `package.json`).
- **Missing `.nuxt` files** – run `pnpm dev` or `pnpm build` to generate the `.nuxt` directory.
- **API failures** – check the browser console for CORS or rate‑limit messages; the composable logs errors.
